# tcpforwarder
A simple TCP Forwarder

## Installation
```bash
npm install -g tcpforwarder
```

## Usage
```
Usage: tcpforwarder <port> <target:port>

Options:
-v --verbose | Verbose any send & received traffic by Client & Server
```
